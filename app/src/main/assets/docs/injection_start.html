شروع مراحل تزریق!

این قسمت شامل راهنمایی‌هایی جهت تزریق داروی سینورا® (آدالیمومب) است.

 

قبل از تزریق

اطمینان حاصل کنید که راهنمای آموزشی مصرف دارو را مطالعه کرده‌اید.

فیزیوجکت™ برای تزریق راحت توسط خودتان طراحی شده است، اما برای تزریق بار اول، آموزش‌های لازم را از پزشک، داروساز یا پرستار خود درخواست کنید. همچنین می‌توانید برای آموزش رایگان تزریق به یکی از مراکز ارکیدلایف مراجعه فرمایید.
